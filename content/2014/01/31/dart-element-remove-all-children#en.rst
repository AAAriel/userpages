[Dart] DOM Element Remove All Children
######################################

:tags: Dart, DOM
:category: Dart
:summary: DOM element remove all children in Dart programming language


Question:

  Assume that `elm` is a dom element, how to remove all children elements
  (nodes) of `elm`?

Answer:

.. code-block :: dart

  elm.children.clear();
